---
title: "Tile & TileGroup"
description: "Larger image-capable cards for feature overviews and product landing sections."
---

`<Tile>` is a larger card variant with a prominent image area and hover animation. Use `<TileGroup>` to arrange tiles in a grid. Unlike `<Card>`, tiles are designed for visual feature showcases rather than navigation links.

## Example

<TileGroup cols={2}>
  <Tile title="API Reference" icon="code-simple" href="/api">
    Interactive endpoint pages generated from an OpenAPI spec, with a live Try It console.
  </Tile>
  <Tile title="CLI & Tooling" icon="wrench" href="/guides/cli-reference">
    Scaffold, lint, and check your docs with a single `thally` binary.
  </Tile>
</TileGroup>

```mdx
<TileGroup cols={2}>
  <Tile title="API Reference" icon="code-simple" href="/api">
    Interactive endpoint pages from your OpenAPI spec.
  </Tile>
  <Tile title="CLI & Tooling" icon="wrench" href="/guides/cli-reference">
    One binary to scaffold, lint, and check.
  </Tile>
</TileGroup>
```

## With image

Add a top image to tiles for visual product showcases. The image scales on hover for a subtle interactive effect.

<TileGroup cols={2}>
  <Tile title="Analytics Dashboard" img="/images/image1.jpg" href="/quickstart">
    Monitor active users, conversion rates, and retention in a single view.
  </Tile>
  <Tile title="API Explorer" img="/images/image2.jpg" href="/api">
    Send live requests directly from the docs with your own API key.
  </Tile>
</TileGroup>

```mdx
<TileGroup cols={2}>
  <Tile title="Analytics Dashboard" img="/images/dashboard.svg" href="/quickstart">
    Monitor users and conversion rates.
  </Tile>
  <Tile title="API Explorer" img="/images/explorer.svg" href="/api">
    Send live requests from the docs.

```

## Props

### `<Tile>`

| Prop | Type | Description |
|---|---|---|
| `title` | `string` | Tile heading |
| `href` | `string` | Makes the tile a link |
| `icon` | `string` | Icon name shown next to the title |
| `img` | `string` | Image shown at the top of the tile |

### `<TileGroup>`

| Prop | Type | Default | Description |
|---|---|---|---|
| `cols` | `1 \| 2 \| 3 \| 4` | `2` | Number of columns |
