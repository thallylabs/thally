This page is a starting example. Keep it up to date with notable changes to your
docs or product — each entry uses the `<Update>` component, and the page is also
published as an RSS feed at [`/changelog/rss.xml`](/changelog/rss.xml) that readers
can subscribe to.

<Update label="Live branding" date="2026-06-20">

- **Brand accent color picker** — recolor the live site with a color wheel, presets, or a hex/RGB value, per light and dark mode, with no redeploy.
- **Per-mode logos & favicons** — upload separate light and dark assets; dark falls back to light.

See [Branding and theming](/guides/branding-and-theming) and the [admin dashboard](/guides/admin-dashboard).

</Update>

<Update label="Admin dashboard" date="2026-05-15">

- A built-in **admin dashboard** at `/admin` controls branding, feature toggles, and team roles, layered on top of the git-committed defaults in `docs.json`.
- Team accounts add Owner / Editor / Viewer roles from a version-controlled roster — no database.

</Update>

<Update label="v0.1.0" date="2025-01-01">

Initial release.

- Next.js App Router foundation with MDX-powered docs
- Responsive shell with keyboard-driven navigation and full-text search
- A component library for callouts, steps, tabs, cards, and API references

</Update>
