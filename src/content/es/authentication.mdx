---
title: Configuración
description: Cómo docs.json controla la navegación, pestañas, grupos y referencias de API.
---

Toda la navegación en Dox está impulsada por un único archivo: `docs.json` en la raíz del proyecto. Este archivo define las pestañas que se muestran en el encabezado, los grupos en cada barra lateral y las páginas dentro de cada grupo.

## Estructura básica

```json
{
  "tabs": [
    {
      "tab": "Overview",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["introduction", "quickstart"]
        }
      ]
    }
  ]
}
```

- **`tabs`** — elementos de navegación de nivel superior que se muestran en el encabezado del sitio
- **`groups`** — secciones de la barra lateral dentro de una pestaña
- **`pages`** — array de slugs de página (relativos a `src/content/`)

## Agregar una pestaña de Referencia de API

Apunta una pestaña a una especificación OpenAPI en lugar de páginas manuales:

```json
{
  "tab": "API Reference",
  "api": {
    "source": "openapi.yaml",
    "defaultGroup": "General"
  }
}
```

Dox analiza la especificación y genera las páginas de endpoints automáticamente.

## Enlaces externos

Agrega una pestaña que enlace a una URL externa:

```json
{
  "tab": "Changelog",
  "href": "/changelog"
}
```
