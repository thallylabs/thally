---
title: "Color"
description: "Muestra una muestra de color con valor hexadecimal — ideal para documentación de marca y tokens de diseño."
---

`<Color>` renderiza un chip de vista previa de color con un nombre y etiqueta hexadecimal opcionales. Úsalo en páginas de directrices de marca o documentación de sistemas de diseño.

## Ejemplo

<Color hex="#10B981" name="Emerald" />
<Color hex="#8B5CF6" name="Violet" />
<Color hex="#F59E0B" name="Amber" />
<Color hex="#EF4444" name="Rose" />

```mdx
<Color hex="#10B981" name="Emerald" />
<Color hex="#8B5CF6" name="Violet" />
<Color hex="#F59E0B" name="Amber" />
<Color hex="#EF4444" name="Rose" />
```

## En una cuadrícula

Usa `<Columns>` para organizar múltiples muestras una al lado de la otra:

<Columns cols={4}>
  <Color hex="#10B981" name="Emerald" />
  <Color hex="#8B5CF6" name="Violet" />
  <Color hex="#F59E0B" name="Amber" />
  <Color hex="#EF4444" name="Rose" />
</Columns>

```mdx
<Columns cols={4}>
  <Color hex="#10B981" name="Emerald" />
  <Color hex="#8B5CF6" name="Violet" />
  <Color hex="#F59E0B" name="Amber" />
  <Color hex="#EF4444" name="Rose" />
</Columns>
```

## Props

| Prop | Tipo | Descripción |
|---|---|---|
| `hex` | `string` | Valor de color hexadecimal incluyendo `#` (p. ej. `#10B981`) |
| `name` | `string` | Nombre de visualización opcional que aparece encima del código hexadecimal |
