# `@thallylabs/migrate`

The shared migration engine used by the Thally CLI, MCP server, and Thally
Cloud. It imports documentation repositories and public documentation URLs
into one canonical bundle before any caller writes files.

Repository migration includes dedicated Mintlify and Docusaurus adapters.
Mintlify configuration is projected from `docs.json`/`mint.json`. Docusaurus
sidebars are parsed as bounded, data-only JSON5—never executed—and preserve
explicit categories, autogenerated slices, document IDs/slugs, positions,
generated indexes, admonitions, tabs, and `static/` assets where supported.
